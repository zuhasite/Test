Installation

To get started with the ZuhaWorldSite, follow these steps:

  1. Clone the repository: 
    git clone https://github.com/zuhasite/test.git
  2. Navigate to the project directory: 
    cd test
  3. Install dependencies: 
    npm install
  4. Run the project: 
    npm start
